A minimal, single-page website listing contact information of Bangladesh IT companies for job seekers.

Visit the website: [https://badhon495.github.io/just-apply/](https://badhon495.github.io/just-apply/)

## Features

- Searchable list of 50+ Bangladesh IT companies
- Click any company name to expand its details
- Shows location, website, career page, email, LinkedIn, and contact person
- No dependencies — open `index.html` directly in any browser

## Usage

Open `index.html` in your browser. No server or build step required.
